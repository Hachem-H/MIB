# Mathematics Is Beautiful

> Pure mathematics is, in it's way, the poetry of logical ideas. - Albert Einstein

As a mathematican, I often feel the need to show the beauty of mathematics, and what better way to do so using visualizations. Using the [processing development environment](https://processing.org/), I coded up a series of representations of a number of mathematical systems and ideas. The main premise of this repository is to contain a set of tools helping me to create youtube visualizations to make certain maths concepts more intuitive to grasp. And for the curious folks out there, I specifically decided to use processing as it is a great teaching tool making coding far less intimidating, and so, anyone can look up the code for a certain topic, make changes and just straight out experiment with ideas to make sure they truely understand what is going on.

Each subject is located in its little folder named accordingly containing the source code and a little `README.md` describing what the visualization is about and explaining a little bit on how the systems involved work.
